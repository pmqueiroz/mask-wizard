## Getting Started

### Installation
Get started by **installing Mask Wizard**.

Run

```shell
npm i mask-wizard

#or

yarn add mask-wizard
```
### Importing

// Js
```js
const maskWizard = require('mask-wizard')
```

// Ts
```ts
import maskWizard from 'mask-wizard'
```
